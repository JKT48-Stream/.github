## About

JKT48 Stream is an unofficial, fan-made web platform built to bring video and live streaming content related to JKT48 together in a single, unified viewing experience. Inspired by the familiar layout and functionality of YouTube, the platform is designed specifically around JKT48 content, allowing fans to browse, watch, and organize videos from multiple official JKT48 channels without needing to switch between platforms.

The project began as a fan initiative and continues to be developed with the goal of giving the JKT48 fan community a dedicated, easy-to-use space to stay up to date with the group's content.

## Content Channels

JKT48 Stream organizes its content into four dedicated channels, each reflecting a different side of JKT48's official content:

| Channel | Description |
|---|---|
| JKT48 | Professional and official content, including music videos and group announcements |
| JKT48TV | Casual, entertainment-focused content such as behind-the-scenes footage, vlogs, and other lighthearted segments |
| JKT48 Live | Live broadcast content streamed directly from JKT48's official live channel |
| IDN Live | Replay recordings of live streams originally broadcast by JKT48 members on the IDN Live platform |

## Core Features

| Feature | Description |
|---|---|
| Multi-Channel Library | Browse content from four official JKT48-related channels in one place |
| Playlists | Create and manage custom playlists from available videos |
| Saved Videos | Save videos to revisit later |
| Watch History | Automatically track previously watched videos |
| Theme Customization | Switch between display themes to personalize the viewing experience |
| No Account Required | Browse and watch content freely without signing up or logging in |

---

## Current Limitations

As JKT48 Stream is an actively developing fan project, the following features commonly found on platforms like YouTube are not yet supported:

| Limitation | Status |
|---|---|
| User Login / Accounts | Not available |
| Commenting | Not available |
| Viewing Comments | Not available |
| Uploading Videos or Photos | Not available |

---

## Data Sources

JKT48 Stream retrieves its content through the following sources:

| Source | Purpose |
|---|---|
| YouTube Data API | Used to fetch video and channel data from official JKT48 YouTube channels |
| JKT48Connect API (Free Version) | An unofficial, community-built API used to source additional JKT48-related data |
